### PROJECT DESCRIPTION:

This project analyzes retail customer shopping behavior from a 3.9K transactional dataset, ranging from different product categories (e.g., Clothing, Footwear, Outerwear, Accessories, etc.). The goal is to gain insights into customers’ spending patterns, product preferences, segments, and subscription behavior to guide strategic business decisions.

---

### CODE AND RESOURCES:

**Python Version:** 3.14 <br>
**Packages and Tools:** Pandas, Jupyter Notebook, PostgreSQL, pgAdmin 4, Google Docs, Google Slides <br>
**Dataset:** [Retail Customer Trends Dataset](https://github.com/amlanmohanty1/customer-trends-data-analysis-SQL-Python-PowerBI/blob/main/customer_shopping_behavior.csv)

---

### FILE DESCRIPTIONS:

**Customer_Behavior_Dataset_Analysis.ipynb:** This file loads in the customer dataset, performs exploratory data analysis (EDA), and feature engineering. It also connects the [newly-processed] dataset to a PostgreSQL server so that we can execute our SQL business questions. More information about the specifics of the EDA and feature engineering can be viewed in the notebook itself. <br><br>
**SQL Business Questions:** Contains the business questions we created and wanted to investigate, as well as the SQL code to answer those questions. <br><br>
**Retail Customer Behavior Dashboard:** An interactive dashboard we created using Power BI to visually represent our data analysis. Includes information such as total number of customers, average purchase amount, average review rating, etc. The dashboard can be filtered to show results for men vs. women, subscribed vs. not subscribed, different categories, and more. <br><br>
**Retail Customer Behavior Analysis:** A report created to show everything we created (i.e., steps to complete this project and our findings). It includes a project overview, a dataset overview, EDA, the SQL business questions, the dashboard, and final business recommendations. <br><br>
**Retail Customer Behavior Analysis Presentation:** A PowerPoint presentation made for stakeholders and clients to communicate our data analysis, findings, and overall business recommendations.
